# MCAN-Based CAN Transmitter on AURIX TC375
A periodic CAN transmitter on Infineon AURIX TC375 microcontroller using MCAN and FreeRTOS, which transmits ADC data as well as displays it in real-time on 7-segment display using DMA and GPT12 timer.
## Project Directory
The worksapce WS_AURIX_Project_4-1_CAN_Tx_ADC can be opened and debugged on Aurix Development Studio.
